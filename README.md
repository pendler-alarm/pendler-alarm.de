# pendler-alarm.de

This is a web app providing a commuter alarm.

## ✨ inspired by

- 🗓️ [Anschluss erreichen Hackathon](https://data.deutschebahn.com/opendata/Anschluss-erreichen-Hackathon-13704734)
- 📁 [Calendr](https://github.com/cbrasser/calendr)
- [🧑 Abdullahi Fatola](https://github.com/AbdullahiFatola): Draft of a searching app
- [🧑 Johannes Reinholz](https://github.com/jreinholz-code): idea of a pendler app
- [🧑 Robert Willemelis](https://github.com/willi84): connection of calendar and routing

## 📄 important files

- constant values: `src/utils/constants/*`
- components: `src/components/*`
- views: `src/views/*`
- router: `src/router/*`
- i18n (language): `src/i18n/*`
- docs: `docs`
- scripts (e.g. for release): `scripts/*`

## used ressources/tools

- [OpenAI](https://openai.com/) with ChatGPT & CODEX
- [Google Calendar API](https://console.cloud.google.com/)
- [openStation API Bahn](https://developers.deutschebahn.com/db-api-marketplace/apis/product/open-station)
- [Vercel for hosting](https://vercel.com/)
- [Vue 3](https://vuejs.org/) with [Vite](https://vitejs.dev/) and [Typescript](https://www.typescriptlang.org/)
- [transitious](https://transitious.org/) for routing with [MOTIS 2 API](https://transitous.org/api/)
- [i18n internationalization](https://vue-i18n.intlify.dev/) Plugin
- [Unsplash for images](https://unsplash.com/) and [Material Design Icons](https://fonts.google.com/icons) for icons
